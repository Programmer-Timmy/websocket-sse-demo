Installatie & Gebruik

Vereisten

  • Node.js (v18 of hoger)
  • npm of yarn

Stappen

1. Clone de repository

git clone https://github.com/Programmer-Timmy/websocket-sse-demo.git
cd websocket-sse-demo

2. Installeer dependencies

npm install

3. Start de backend server (Terminal 1)

npm run server

Server draait op http://localhost:3001

4. Start de frontend (Terminal 2)

npm run dev

Frontend draait op http://localhost:5173
